### Stack used
This app uses Vite + SWC. A minimal setup to get React working in Vite with HMR and some ESLint rules.

The weather services are provided by [Free Weather Api](https://www.weatherapi.com/).

### Dependencies

- Axios
- Framer Motion
- Free Weather Api
- TailwindCSS
- Vite
- React Hook Form
- React Hot Toast
- Zustand
